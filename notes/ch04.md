# Chapter 04 Our First Program

## 4.1 Comments

三种注释！

```c++
int main()
{
    // this is a comment
}
```

```c++
int main()
{
    // this is a comment
    // this is another comment
}
```

```c++
int main()
{
    /* This is a
    multi-line comment */
}
```
